# Multi-Agent 协同开发方法论

> 适用于任何想通过 OpenClaw 搭建多 Agent 自动化开发流水线的工程师。
> 与具体项目完全解耦，可直接复用于任何技术栈。

## 文档说明

| 文件 | 内容 | 适用人群 |
|------|------|---------|
| [METHODOLOGY.md](./METHODOLOGY.md) | 核心方法论（原则、模式、标准） | 架构师/决策者 |
| [PLAYBOOK.md](./PLAYBOOK.md) | 操作手册（可直接发给 OpenClaw 执行） | 开发者 |
| [CASE_STUDY.md](./CASE_STUDY.md) | 实战案例（sanitary-admin 项目验证） | 参考学习 |

## 快速开始

1. 将 `PLAYBOOK.md` 全文发给你的 OpenClaw
2. OpenClaw 会引导你完成流水线搭建
3. 参考 `CASE_STUDY.md` 了解实际效果
